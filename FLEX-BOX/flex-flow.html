Usamos { row nowrap }:

Se tirarmos o no, provavelmente não teremos alteraçoes:

839 1
736 2
862 3
826 4

Não teremos alteraçoes quando se editar o width.

Usamos o { row wrap-reverse }:

839 1
736 2
862 3
826 4

Usamos { column wrap }:

839 1
736 2
862 3
826 4

Usamos { column wrap-reverse }:

839 1
736 2
862 3
826 4

Apenas teste:

As alteraçoes são validas em { "row-column-reverse" e wrap-reverse }:

839 1
736 2
862 3
826 4
